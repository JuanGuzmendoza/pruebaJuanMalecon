Aquí tienes todas las tablas de tu base de datos, con cada campo, tipo de dato y número de caracteres (cuando aplica):

Sensors
sensor_id : INT (PK)
sensor_code : VARCHAR(30)
type_sensor : VARCHAR(200)
price : DECIMAL(12,2)
watering : DATETIME
irrigation_system : VARCHAR(100)
sensor_status : VARCHAR(100)
maintenance : DATE
created_at : TIMESTAMP
updated_at : TIMESTAMP


Technicians
technical_id : INT (PK)
name : VARCHAR(200)
created_at : TIMESTAMP
updated_at : TIMESTAMP

Crops
crop_id : INT (PK)
type_crop : VARCHAR(200)
type_soil : VARCHAR(100)
fertilizer : VARCHAR(250)
it_organic : BOOLEAN
crop_variety : VARCHAR(205)
tons_produced : DECIMAL(12,2)
sensor_id : INT (FK)
created_at : TIMESTAMP
updated_at : TIMESTAM
updated_at : TIMESTAMP


Farms
farm_id : INT (PK)
name : VARCHAR(200)
region : VARCHAR(200)
created_at : TIMESTAMP
updated_at : TIMESTAMP
technical_id : INT (FK)
crop_id : INT (FK, puede ser NULL por ON DELETE SET NULL)
Leyenda:

(PK): Primary Key
(FK): Foreign Key
VARCHAR(n): Cadena de texto de longitud n
INT: Entero
DECIMAL(12,2): Número decimal con 12 dígitos y 2 decimales
BOOLEAN: Verdadero/Falso
TIMESTAMP/DATETIME/DATE: Fechas y horas
